## About

I'm a MERN Stack developer with a focus on backend engineering — designing APIs, modeling data, and reasoning about scale rather than just shipping UI. Right now I'm deepening my DSA and backend scalability skills while building full-stack projects end to end.

- Backend & API design with Node.js, Express, and MongoDB
- Comfortable across the stack: REST APIs, auth (JWT), real-time features (Socket.io)
- Practicing data structures & algorithms consistently
- Open to collaborating on full-stack and open-source projects
